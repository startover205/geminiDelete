## 🇺🇸 English (Default)

### Short Description
Quickly delete Gemini conversations with a custom shortcut and optional one-click sidebar delete button.

### Detailed Description
🔥 Updated for the May 19 Gemini UI changes — Fully compatible!

Delete Shortcut for Gemini™ helps you clean up Gemini chats faster.

Use a custom keyboard shortcut to trigger the delete flow for the active conversation, or turn on a visible delete button in the Gemini sidebar for one-click cleanup.

Features:
- Set your own delete shortcut
- Delete the active Gemini conversation without digging through menus
- Optionally show a delete button next to conversations in the chat list
- 🌍 Full support for 9 languages with an easy-to-use in-app language switcher!
- Store settings locally using Chrome storage
- Lightweight extension with no analytics or tracking

Privacy:
- Stores only your extension settings, such as shortcut preferences and UI toggles
- Does not collect or transmit Gemini conversation content to the developer
- Does not include analytics or ad tracking
