Privacy Policy

Delete Shortcut for Gemini™ Privacy Policy

Delete Shortcut for Gemini™ is a Chrome extension that helps users quickly delete Gemini conversations using a shortcut and optional UI controls.

Last updated: March 31, 2026

What this extension stores

Delete Shortcut for Gemini™ stores only the settings needed for the extension to work, such as your chosen keyboard shortcut and UI preferences.

  • Custom shortcut settings
  • Whether the delete button is shown in the Gemini conversation list

What this extension does not collect

Delete Shortcut for Gemini™ does not collect, sell, or transmit your Gemini conversation content to the developer.

  • No chat messages are sent to the developer
  • No analytics or tracking scripts are used by the extension itself
  • No personal data is sold or shared for advertising purposes

How data is used

Stored settings are used only to provide the extension’s functionality on your device and across your signed-in Chrome profile when Chrome sync is enabled.

Third-party links

The popup includes links to third-party services:

  • A Tally feedback form
  • A support page hosted on GitHub Pages
  • The project GitHub repository

When you open those links, their own privacy policies and terms apply.

Permissions

Delete Shortcut for Gemini™ currently uses the storage permission to save extension settings. It runs on https://gemini.google.com/* to provide delete-related UI and shortcut behavior.

Contact

If you have questions about this policy, please open an issue on GitHub.
